* Preamble
To analyze the flight logs, execute ~cmpLogs.py~

1) Create a virtual environment and activate it
2) Install the requirements
3) Run the script

#+begin_src bash
  python -m venv log-venv
  source log-venv/bin/activate
  pip install -r requirements.txt
  python cmpLogs.py
#+end_src
